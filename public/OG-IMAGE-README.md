# Open Graph Image

## Required

Create an image file named `og-image.png` in this directory.

### Specifications:
- **Filename**: `og-image.png`
- **Dimensions**: 1200 x 630 pixels
- **Format**: PNG or JPG
- **File size**: < 1MB recommended

### What to include:
- Your site logo
- Site name: "Sekian Info"
- Tagline: "Dashboard Informasi Harian Indonesia"
- Clean, professional design
- High contrast for readability

### Tools to create:
- [Canva](https://www.canva.com/) - Free templates
- [Figma](https://www.figma.com/) - Design tool
- [OG Image Generator](https://og-image.vercel.app/) - Quick generator

### Preview:
This image will appear when your site is shared on:
- Facebook
- Twitter
- LinkedIn
- WhatsApp
- Slack
- Discord
